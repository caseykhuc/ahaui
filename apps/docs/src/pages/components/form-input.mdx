  <li>
    Label  &nbsp;
    <i className="u-text200">- Short descriptive text which corresponds to the information required inside the field.</i>
  </li>
  <li>
    Placeholder&nbsp;
    <i className="u-text200">- The placeholder specifies a short hint that describes the expected value of an input field (i.e. a sample value or a short description of the expected format).</i>
  </li>
  <li>
    Value&nbsp;
    <i className="u-text200">- The value specifies the text information, data or value which is entered by users.</i>
  </li>
  <li>
    Feedback&nbsp;
    <i className="u-text200">- The feedback displays the messages from the system in specific cases i.e. invalid input, valid input, etc.</i>
  </li>
  <li>
    Input border&nbsp;
    <i className="u-text200">- The input border attribute specifies different input states (hovered, invalid, valid, etc.) by colors.</i>
  </li>
  <li>
    Input background&nbsp;
    <i className="u-text200">- The standard input background is white. It will only be changed to grey when the field is disabled.</i>
  </li>
</ol>
